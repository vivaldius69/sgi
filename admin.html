관리자 사용 안내서

1 첫 로그인 & 비밀번호 변경

  • 초기 비밀번호는 0000 입니다.
  • 처음 로그인 후 반드시 비밀번호 변경을 해주세요.
  • 상단 우측 비밀번호 변경 버튼 클릭 → 현재 비밀번호 입력 → 새 비밀번호 입력(4자리 이상) → 확인.
  • ⚠️ 비밀번호는 브라우저 저장소에 보관됩니다. 같은 기기·브라우저에서만 유효합니다.

2 모임 개설 방법

  • 모임 개설 탭을 클릭합니다.
  • 모임 주제, 날짜, 최대 인원(필수), 장소, 설명을 입력합니다.
  • 모임 개설 버튼 클릭 → 게시판에 즉시 등록됩니다.
  • 참가자는 참가자 신청 페이지(user.html)에서 모임을 보고 신청합니다.

3 참석자 명단 확인

  • 모임 게시판 탭에서 원하는 모임 카드를 클릭합니다.
  • 하단에 참석자 명단 패널이 열립니다.
  • 명단은 가나다순(이름순)으로 자동 정렬됩니다.
  • 이름이나 전화번호로 실시간 검색이 가능합니다.

4 모임 상태 변경 (모집중 ↔ 마감)

  • 모임 카드에서 마감 처리 버튼을 클릭하면 더 이상 신청을 받지 않습니다.
  • 모집 재개를 클릭하면 다시 모집 중 상태로 돌아옵니다.
  • 최대 인원이 다 차면 참가자 페이지에서 자동으로 마감 표시됩니다.

5 인쇄 및 워드 저장

  • 참석자 명단 패널에서 인쇄 버튼 → A4 최적화 인쇄 창이 열립니다.
  • 워드 저장 버튼 → .doc 파일이 컴퓨터에 저장됩니다.
  • 인쇄 시 삭제 버튼 열은 자동으로 숨겨집니다.

6 참가자 삭제

  • 참석자 명단 표에서 각 행의 삭제 버튼 클릭 → 확인 후 삭제됩니다.
  • 삭제 후에는 복구가 불가능합니다. 신중히 사용해 주세요.

7 참가자에게 안내하는 방법

  • 이 사이트를 배포(Publish) 한 뒤 나오는 URL을 참가자에게 공유합니다.
  • 참가자는 홈 화면 → 참가자 신청을 눌러 신청합니다.
  • 관리자 URL(/admin.html)은 외부에 공유하지 않도록 주의해 주세요.

주의 사항

  • 이름·전화번호·계좌번호는 개인정보입니다. 수집 목적 외 사용하지 마세요.
  • 비밀번호 분실 시: 브라우저 개발자 도구에서 localStorage의 adminPassword 키를 삭제하면 초기화(0000)됩니다.
  • 데이터는 서버에 저장되므로 사이트 배포 후에도 유지됩니다.
