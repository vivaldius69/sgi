새 걷기모임 개설

1 관리자 로그인

  • 관리자 페이지는 메인 화면에 노출되지 않습니다.
  • 운영자는 /admin.html 주소로 직접 접속합니다.
  • Supabase Authentication에 생성한 관리자 이메일과 비밀번호로 로그인합니다.

2 새 걷기모임 개설

  • 모임 개설 탭에서 제목, 날짜, 시간, 최대 인원을 입력합니다.
  • 장소와 설명을 함께 적으면 참가자 페이지에서 더 친절하게 안내할 수 있습니다.
  • 개설된 모임은 참가자 페이지에 즉시 반영됩니다.

3 참가자 명단 관리

  • 모임 게시판에서 원하는 카드의 명단 보기를 누르면 참가자 목록이 열립니다.
  • 참가자 명단은 이름 기준 가나다순으로 정렬됩니다.
  • 검색창에서 이름이나 전화번호를 입력해 빠르게 찾을 수 있습니다.

4 인쇄 및 워드 저장

  • 인쇄 버튼은 A4 용지 기준 미리보기를 엽니다.
  • 워드 저장 버튼은 .doc 파일로 참가자 명단을 저장합니다.
  • 인쇄 시 삭제 버튼 열은 자동으로 숨겨집니다.

운영 주의사항

  • 전화번호와 계좌번호는 개인정보입니다. 운영 목적 외 사용하지 마세요.
  • 관리자 계정 비밀번호는 Supabase Authentication에서 별도로 안전하게 관리하세요.
  • 권한 정책(RLS)을 README 안내대로 정확히 설정해야 사이트가 정상 동작합니다.
