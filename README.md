# 모임 모집 시스템

스마트폰·PC 모두에서 사용 가능한 모임 참가 신청 및 관리 웹 애플리케이션입니다.

---

## 주요 기능

### 참가자(사용자) 기능
- 현재 모집 중인 모임 목록 확인
- 이름·전화번호·계좌번호 입력 후 신청
- 중복 신청 자동 방지 / 최대 인원 초과 차단

### 관리자 기능
- 비밀번호 보호 로그인 (초기값 `0000`, 변경 가능)
- 모임 개설(주제, 날짜, 장소, 최대 인원, 설명)
- 게시판형 모임 목록 조회 (전체 / 모집중 / 마감 필터)
- 모임 클릭 → 참석자 명단 패널 (가나다순 정렬)
- 실시간 이름·전화번호 검색
- 모집중 ↔ 마감 상태 토글
- 참가자 개별 삭제
- 참석자 명단 인쇄 (A4 최적화)
- 참석자 명단 워드(.doc) 다운로드
- 관리자용 사용 안내서 내장 (탭 내 확인 가능)

---

## 파일 구조

```
index.html      홈 (사용자/관리자 모드 선택)
user.html       참가자 신청 페이지
admin.html      관리자 관리 페이지
css/style.css   공통 스타일 (모바일 반응형)
README.md       프로젝트 문서
```

---

## 접속 방법

| 대상 | URL |
|------|-----|
| 참가자 | `/` 또는 `/index.html` → "참가자 신청" 선택 |
| 관리자 | `/admin.html` → 비밀번호 입력 |

---

## 관리자 초기 비밀번호

`0000`  
→ 로그인 후 상단 **비밀번호 변경** 버튼으로 즉시 변경하세요.

---

## 데이터 모델

### meetings 테이블
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 ID |
| title | text | 모임 주제 |
| date | text | 날짜 |
| location | text | 장소 |
| maxParticipants | number | 최대 인원 |
| description | rich_text | 설명 |
| status | text | 모집중/마감 |
| createdAt | datetime | 생성 일시 |

### participants 테이블
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 ID |
| meetingId | text | 모임 ID (외래키) |
| name | text | 참가자 이름 |
| phone | text | 전화번호 |
| account | text | 계좌번호 |
| appliedAt | datetime | 신청 일시 |

## 배포

Publish 탭에서 발행하면 즉시 공개됩니다.  
배포 후 URL을 참가자에게 공유하세요.

---

## 주의 사항

- 관리자 비밀번호는 브라우저 `localStorage`에 저장됩니다. 같은 기기·브라우저에서만 유효합니다.
- 이름·전화번호·계좌번호는 개인정보입니다. 수집 목적 외 사용하지 마세요.
- 비밀번호 분실 시: 브라우저 개발자 도구 → Application → LocalStorage → `adminPassword` 키 삭제 → 초기값(`0000`)으로 복원.
